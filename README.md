# 🌿 CropMitra – Smart Irrigation System

## 📌 Overview
CropMitra is an IoT-based smart irrigation system that monitors soil moisture, temperature, and humidity using NodeMCU ESP8266 and sensors. It automatically controls a water pump based on real-time data to optimize water usage, reduce wastage, and improve crop yield.

## 🚜 Problem Statement
Traditional irrigation methods often lead to overwatering or underwatering, causing water wastage and low crop productivity. This project provides an automated and efficient irrigation solution.

## 🛠 Technologies Used
- NodeMCU ESP8266
- Soil Moisture Sensor
- DHT11 Sensor
- Relay Module
- Arduino IDE
- HTML, CSS, JavaScript

## ⚙️ Features
- Real-time soil monitoring
- Automatic pump control
- Water conservation
- IoT-based system

## 🚀 Working Principle
1. Sensors collect soil moisture and temperature data.
2. If moisture is below the set threshold, the pump turns ON.
3. If moisture is adequate, the pump turns OFF.
4. The process repeats continuously for efficient irrigation.
